# Cookery App

## Installation

* This app requires at least ruby 2.0.
* _ImageMagick_ is also required. It has to be installed
  independently.

## The big list of technologies used

* Rails 4

### Frontend

* HAML
* SASS
* Bootstrap
* Paperclip

### Backend

 * FactoryGirl
 * closure_tree

### Tests

 * rspec
 * guard
